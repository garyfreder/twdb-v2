This web service is designed to work with App Inventor for All and the TinyWebDB component. The end-goal of this service is to communicate with a mobile app created with App Inventor.

You can use it to give you a twdb server and to come up with your own server!

The page your are looking at is a web page interface to the web service to help programmers with debugging. You can log in/out, leave a message in the guestbook and go to the twdb page and invoke the get and store operations by hand, view the existing entries and also delete individual entries.

This server is based on the server Google/MIT built. It is updated to use Python 2.7, it uses templates and you can use it to build your own server.

More info on the Jefferson Software's twdb info page.
